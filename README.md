Apache Impala Kubernetes Operator based on Operator framework SDK.

|-------|---------------|-------------------|

| Level 1 | Basic Install     | :heavy_check_mark: |
| Level 2 | Seamless Upgrades | :white_check_mark: |
| Level 3 | Full Lifecycle    | :white_check_mark: |
| Level 4 | Deep Insights     | :white_check_mark: |
| Level 5 | Auto Pilot        | :white_check_mark: |

## Description
The Impala Operator manages Impala clusters deployed to Kubernetes 
and automates tasks related to operating a Impala cluster. It provides 
a full management life-cycle for Impala, including deployment, upgrades, 
scaling, and configuration changes.
